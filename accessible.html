Accessibility Cheatsheet

What is Web Accessibility?

Web accessibility means that people with disabilities can use the web equally.

Web Accessibility Guidelines

Section 508

Section 508 is a federal law that requires all electronic and information technology developed, procured, maintained, or used by the federal government be accessible to people with disabilities.

WCAG 2.1

The Web Content Accessibility Guidelines are a series of internationally recognized and adopted standards produced by the World Wide Web Consortium (W3C) for improving accessibility.

Pour Principles

Perceivable
Users must be able to perceive the information being presented
Operable
Users must be able to operate the interface
Understandable
Users must be able to understand the information as well as the operation of the user interface
Robust
Users must be able to access the content as technologies advance
Conformance Levels
Level Conformance Type Description
A Minimum The most basic web accessibility features
AA Mid-range Deals with the biggest and most common barriers
AAA Highest The highest level of web accessibility
